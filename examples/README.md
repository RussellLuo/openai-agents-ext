# Examples

This directory contains two examples of how to use the `openai-agents-ext` library.


## Quickstart

Run the `stream_tools.py` example:

```bash
cd examples
export OPENAI_API_KEY=<your_api_key>
python stream_tools.py
```

Run the `agents_as_stream_tools.py` example:

```bash
cd examples
export OPENAI_API_KEY=<your_api_key>
python agents_as_stream_tools.py
```
